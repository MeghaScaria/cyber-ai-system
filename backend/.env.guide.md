# Backend Environment Setup Guide

To run the Cyber AI System backend, you need to configure several external services. Follow these steps to create your `.env` file.

## 1. Local Environment File
Create a file named `.env` in the `backend/` directory and copy the contents from `.env.example`.

## 2. MongoDB Setup
- **Option A (Local)**: Install [MongoDB Community Edition](https://www.mongodb.com/try/download/community).
  - `MONGODB_URL="mongodb://localhost:27017"`
- **Option B (Cloud)**: Create a free cluster on [MongoDB Atlas](https://www.mongodb.com/cloud/atlas).
  - Replace the URL with your Atlas connection string.

## 3. Firebase Admin SDK (Notifications)
- Go to the [Firebase Console](https://console.firebase.google.com/).
- Select your project (or create one).
- Navigate to **Project Settings > Service Accounts**.
- Click **Generate New Private Key**. A JSON file will download.
- Place this JSON file in the `backend/` folder and name it `firebase-adminsdk.json`.
- In `.env`: `FIREBASE_CREDENTIALS_PATH="firebase-adminsdk.json"`

## 4. VirusTotal API (URL Analysis)
- Sign up at [VirusTotal](https://www.virustotal.com/).
- Go to your profile and find **API Key**.
- In `.env`: `VIRUSTOTAL_API_KEY="your_api_key"`

## 5. Google Safe Browsing API
- Go to [Google Cloud Console](https://console.cloud.google.com/).
- Create a new project or select an existing one.
- Go to **APIs & Services > Library** and search for "Safe Browsing API". Enable it.
- Go to **APIs & Services > Credentials** and click **Create Credentials > API Key**.
- In `.env`: `GOOGLE_SAFE_BROWSING_API_KEY="your_api_key"`

## 6. Running the Server
Once the `.env` is set up, install dependencies and start the server:
```powershell
cd backend
pip install -r requirements.txt
uvicorn app.main:app --reload
```
